# Guide Acheteur GoldLink

L'acheteur ou locataire peut :
- parcourir le catalogue
- filtrer les bijoux par type, purete, prix et localisation
- consulter les fiches detaillees
- contacter un vendeur
- reserver un bijou propose a la location
- acheter un bijou propose a la vente
- consulter son tableau de bord

Bonnes pratiques pour l'acheteur :
- verifier le type de transaction disponible : location, vente ou les deux
- lire la description complete avant de reserver
- confirmer les dates de location avec le vendeur
- verifier la purete et le poids du bijou
- consulter les avis et la note du vendeur

Avant de reserver un bijou :
- verifier la disponibilite
- verifier le prix journalier
- verifier le depot de garantie si applicable
- poser les questions importantes via la messagerie

Le chatbot doit orienter l'acheteur vers les informations disponibles sans promettre des elements absents du systeme.
